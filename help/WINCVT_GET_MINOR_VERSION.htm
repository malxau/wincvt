WINCVT_GET_MINOR_VERSION

The WINCVT_GET_MINOR_VERSION macro retrieves the minor version component from a version DWORD.

DWORD
WINCVT_GET_MINOR_VERSION(
  DWORD Version     // The version to extract from
);

Parameters

Version
Specifies the version whose minor component is required. This version can be obtained from the WinCvtHeaderVersion or WinCvtLibraryVersion function.

Return Values

The minor version component.

Remarks

The WINCVT_GET_MINOR_VERSION macro retrieves the minor version component from a version DWORD. A version is represented as "major.minor.micro". To obtain the major version component, use the WINCVT_GET_MAJOR_VERSION macro. To obtain the micro version component, use the WINCVT_GET_MICRO_VERSION macro.
