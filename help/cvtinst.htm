cvtinst.exe

The cvtinst.exe binary installs a converter file into the registry ready for use by applications.

cvtinst.exe [-e] [-u] <converter>

Parameters

-e
Uninstalls, rather than installs, the specified converter. This switch is available in WinCvt 0.3.0 or newer.
-u
Performs the installation or uninstallation on the user registry, rather than the system-wide registry. Note that not all applications will process data in the user registry. This switch is available in WinCvt 0.3.0 or newer.
converter
Full path to the converter file to install or uninstall, eg. "C:\converters\foo.cvt"

Return Values

The cvtinst.exe binary returns 0 on success and nonzero on failure.

Remarks

The cvtinst.exe binary installs a converter file into the registry ready for use by applications.

QuickInfo
