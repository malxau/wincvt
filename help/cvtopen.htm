cvtopen.exe

The cvtopen.exe binary converts a file from its current format to RTF for modification, and converts back to its original format when modifications are complete.

cvtopen.exe SourceFile [Application]

Parameters

SourceFile
Specifies the original file to edit. This file can be in any format for which converters exist.
Application
Specifies the application to use for modifications. If not specified, the default system application for modifying RTF files will be used.

Return Values

The cvtopen.exe binary returns 0 on success and nonzero on failure.

Remarks

The cvtopen.exe binary converts a file from its current format to RTF for modification, and converts back to its original format when modifications are complete.
