cvttype.exe

The cvttype.exe binary queries the system registry for a converter capable of converting the specified document.

cvttype.exe [-q] file

Parameters

-q
Quick mode. In this mode, the registry is investigated for a converter capable of reading the document by extension only. If this flag is not specified, the file is actively interrogated to determine if it can be converted.
file
Specifies the file whose type is to be determined.

Return Values

The cvttype.exe binary returns 0 on success and nonzero on failure.

Remarks

The cvttype.exe binary queries the system registry for a converter capable of converting the specified document.

QuickInfo

WinCvt: Requires 0.1.0.
