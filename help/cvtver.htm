cvtver.exe

The cvtver.exe binary displays the version of the WinCvt library in use and the version of WinCvt that it was compiled against.

cvtver.exe

Parameters

The cvtver.exe binary accepts no parameters.

Return Values

The cvtver.exe binary returns 0 on success and nonzero on failure.

Remarks

The cvtver.exe binary displays the version of the WinCvt library in use and the version of WinCvt that it was compiled against.

QuickInfo

WinCvt: Requires 0.1.0.
