cvtview.exe

The cvtview.exe binary displays a document in any format for browsing.

cvtview.exe [filename]

Parameters

filename
If specified, the file to load when the application starts.

Return Values

The cvtview.exe binary returns 0 on success and nonzero on failure.

Remarks

The cvtview.exe binary displays a document in any format for browsing.

QuickInfo

WinCvt: Requires 0.3.0.
